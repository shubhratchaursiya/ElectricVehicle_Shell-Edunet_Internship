# ElectricVehicle_Shell-Edunet_Internship

This repository contains work from **Week 1** of my internship on **Electric Vehicles (EVs)** under the Shell-Edunet program. It explores a dataset of EV charging stations and aims to deliver insight into the EV ecosystem through data cleaning, analysis, and visualization.
